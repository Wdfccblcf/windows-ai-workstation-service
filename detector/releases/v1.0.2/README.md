# Windows AI 自动检测器（检测专用版）

这是只读检测版本，不包含修复引擎、软件安装、系统功能修改或管理员权限申请。

## 使用方法

1. 仅支持 Windows 11。
2. 解压完整文件夹，不要单独复制其中一个脚本。
3. 双击 `Start-Windows-AI-Scan.cmd`。
4. 点击“开始19项自动检测”。
5. 检测结束后查看 `windows-ai-scan` 目录中的报告。

## 输出

- `audit-progress.jsonl`：19项脱敏进度事件；
- `audit.json`：结构化检测结果；
- `audit-report.md`：用户可读报告；
- `audit.log`：脱敏日志。

## 安全边界

- 不上传数据，不联网下载，不创建 API Key；
- 不读取或显示密钥值、完整用户名、设备序列号或完整用户路径；
- 不安装、卸载、删除、修改 PATH、修改注册表、启用 WSL、启动或重置 Docker；
- 检测结果不等于修复承诺；任何修复必须另行在平台固定范围，并由客户本人全程在场确认。

启动时会校验 `SHA256SUMS.txt`。校验失败会拒绝运行。由于本包尚未使用商业代码签名证书，Windows 可能显示未知发布者；不得绕过 Windows 安全拦截，也不得从非官方来源接收修改版。
