常见问题

下单前，先把顾虑问完

01会不会看到我的密码或 API Key？＋

不会要求你发送密码、验证码或密钥。需要输入时由你本人操作；Strict 体检只记录“存在/不存在”，不记录变量值。

02是不是下单后什么报错都包修？＋

不是。下单前先确认固定范围。网络封锁、第三方账号或额度、公司管控设备、Docker 深度数据损坏属于外部阻塞，报告会写明剩余步骤。

03远程时会操作我的私人文件吗？＋

不会主动打开与本次安装无关的文件。客户全程在场；管理员授权、重启和系统功能变更都要单独确认。

04为什么只支持 Windows 11？＋

首期先把一个系统做稳。Windows 10 会在脚本与远程流程完成验证后再开放。

05可以私下转账更便宜吗？＋

不可以。所有订单和退款均走平台担保流程，页面不放微信、二维码或站外付款方式。

06修不好怎么办？＋

若范围内操作因外部条件阻塞，仍交付已完成的体检、通过项、未通过项、阻塞原因和可执行的后续步骤；取消或退款按平台规则处理。
